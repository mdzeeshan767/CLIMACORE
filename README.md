ClimaCore is a web-based temperature monitoring and alert system designed to track, analyze, and manage temperature data across multiple locations in real time.
-------------------The system allows users to:-
=Record temperature readings.
=Monitor live environmental conditions.
=Visualize historical temperature data through interactive dashboards.
=Receive automated alerts when temperature exceeds predefined thresholds.

----------------------------Technology Stack-----------------------------------

Frontend: HTML, CSS, JavaScript.
Backend: Python (Flask framework).
Database: MySQL.

--------------------------------Core Features---------------------------------------

User authentication system.
Role-based access control (administrators, managers, technicians, etc.).
Real-time data visualization using graphical charts.
Automated alert and notification system.
Report generation and analytical reporting tools.
